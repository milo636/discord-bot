# discord-bot

Bot de Discord hecho con discord.js.

## Configurar el token

El token real no va en GitHub. En tu computadora, dentro de la carpeta del bot, crea un archivo llamado `.env` y pon:

```env
DISCORD_TOKEN=ACA_PEGA_EL_TOKEN_DE_TU_BOT
```

El archivo `.env.example` queda como ejemplo seguro para copiar.

## Ejecutar

```bash
npm install
npm start
```
